関数を定義する。

```python
#def function(a, b, c,...)で関数を定義する。関数名（function）や引数（a, b, c,...）は任意。
#return 数式 で引数を使った計算結果を返す。

def bmi(height, weight):
    return weight / (height/100.0)**2
```

実際に身長と体重の数値を入れて計算させてみる。


```python
bmi(188.0, 104.0)
```




    29.425079221367138
